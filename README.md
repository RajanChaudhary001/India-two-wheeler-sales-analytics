# 🏍️ Indian Bike Sales Analysis — Power BI Dashboard

An interactive Power BI dashboard that analyzes the Indian two-wheeler (bike) market, covering pricing trends, fuel type distribution, resale value, regional demand, and manufacturing trends across years.

![Dashboard Preview](dashboard-preview.png)

## 📊 Overview

This project explores bike sales data across India using **Power BI**, providing key business insights through interactive slicers, charts, and KPI cards. The dashboard helps identify pricing patterns, popular brands/models, and regional sales performance for the Indian bike market.

## ✨ Features / Visuals

- **KPI Cards** — Total Price, Total Bikes Sold, Resale Price
- **Average Bike Price by Brand** — Bar chart comparing brand-wise average pricing
- **Fuel Type Distribution** — Pie chart (Electric, Petrol, Hybrid split)
- **Original Price vs Resale Price by Brand** — Comparative brand analysis
- **Manufacture Bike by Year** — Yearly production trend
- **Bike Price Distribution Across City Tiers** — Metro, Tier 1, Tier 2, Tier 3 comparison
- **Top 10 Models** — Best performing bike models
- **Bike Sale by State** — State-wise demand (UP, Tamil Nadu, Rajasthan, etc.)
- **Top 10 Models by Price** — Price-wise model ranking
- **Interactive Filters** — Seller Type, Brand, Registration Year

## 🛠️ Tools & Technologies

- **Power BI Desktop** — Dashboard design & visualization
- **DAX (Data Analysis Expressions)** — Custom measures and calculations
- **Power Query** — Data cleaning and transformation

## 🔍 Key Insights

- Electric and Petrol bikes dominate the fuel type split, followed by Hybrid.
- Bajaj and Yamaha show the highest average pricing among analyzed brands.
- Uttar Pradesh, Tamil Nadu, and Rajasthan are the top-selling states.
- 2021 recorded the highest number of manufactured bikes among the years analyzed.
- Tier 2 cities show notably higher price distribution compared to Metro and Tier 1.

## 🚀 How to Use

1. Clone this repository
   ```bash
   git clone https://github.com/<your-username>/indian-bike-sales-analysis.git
   ```
2. Open `Indian_Bike_Sales_Analysis.pbix` in **Power BI Desktop**
3. Use the slicers (Seller Type, Brand, Registration Year) to explore the data interactively
